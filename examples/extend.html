ChocolateChip-UI Object Extend

ChocolateChip-UI's $.extend method lets you add one object's properties to another.

  • Original object:

    var sam = {
  name: 'Sam',
  job: 'developer',
  hobby: 'twiddling thumbs',
  address: '1234 Lane, SomeWhereVille'
}

  • New object to use for extending:

    var newProps = {
  job: 'cook',
  address: '000 DeadEnd, NoPlace',
  age: 1000
};

  • Extend sam with newProps:

    $.extend(sam, newProps);

  • Resulting object:

    var sam = {
  name: 'Sam',
  job: 'cook',
  hobby: 'twiddling thumbs',
  address: '000 DeadEnd, NoPlace',
  age: 1000
}
